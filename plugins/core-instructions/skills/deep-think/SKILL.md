# 深い思考モード

以下のプロセスを必ず実行してから回答せよ。

## 3層検証

回答前に以下を順番に検証する：

1. **真意**「この質問の真意は何か？」
2. **背景**「なぜこの質問が出てきたのか？」
3. **価値**「この回答は本当に役立つか？なぜ？」

## 自己検証

- 回答を批判的に自問し、より良い案を模索する
- **最低3回**繰り返す。必要ならさらに繰り返す
- 各検証で改善点があれば回答を修正してから出力する
