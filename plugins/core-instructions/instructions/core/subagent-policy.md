# サブエージェント利用ポリシー（YOU MUST）

## 原則

**メインセッションのコンテキストを守れ。** タスク実行は極力サブエージェントに委譲する。
特に出力を得るだけの処理（探索・ビルド・テスト等）は必ずサブエージェントで実行すること。

## モデル選択と報告

起動時に必ず以下の形式で報告する：

```
［モデル選択］Haiku - 探索タスクのため
［モデル選択］Sonnet - 通常の実装タスクのため
［モデル選択］Opus - 複雑なアーキテクチャ設計のため
```

## モデル選択基準

| モデル | コスト比 | 用途 |
|--------|----------|------|
| **Haiku** | x0.04（1/25） | Explore探索、ファイル操作、整形、軽量な確認 |
| **Sonnet** | x0.2（1/5） | 実装、レビュー、リファクタ、テスト・ドキュメント作成、一般調査 |
| **Opus** | x1（基準） | 複雑な設計判断、深いコードベース理解、難解なバグ分析 |

## 判断ルール

1. **Explore エージェントは常に Haiku**
2. **迷ったら Sonnet**
3. **Opus は明確な理由がある場合のみ**（コスト25倍を正当化できるか自問）
